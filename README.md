## 使用方法

1. 安装一个archlinux系统

2. 登录之后从tty运行以下命令
   
   ```bash
   # 1. 安装 git
   sudo pacman -Syu git
    
   # 2. 克隆仓库
   git clone https://github.com/dgxz99/arch-setup.git
   
   # 3. 进入目录并运行
   cd arch-setup
   sudo bash install.sh
   ```
    一条命令版

    ```bash
    sudo pacman -Syu git && git clone https://github.com/dgxz99/arch-setup.git && cd arch-setup && sudo bash install.sh
    ```
